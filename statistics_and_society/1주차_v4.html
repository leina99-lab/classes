각 변수가 수치형(Numerical)인지 범주형(Categorical)인지 파이썬으로 확인해 봅시다.

import pandas as pd # 가상 데이터 생성 data = { 'name': ['Travis', 'Bexar', 'Dallas', 'Harris'], 'income': [50000, 45000, 52000, 48000], 'unemployment_rate': [0.04, 0.05, 0.03, 0.06], 'metro': ['yes', 'yes', 'yes', 'no'] } df = pd.DataFrame(data) # [미션] 데이터의 타입(dtypes)을 출력하여 변수 유형을 확인하세요. print("=== 데이터 미리보기 ===") print(df.head()) print("\n=== 변수 타입 정보 ===") print(df.dtypes)

- int64, float64는 수치형(소득, 비율)입니다.
Exercise 1.3 관찰 연구 vs 실험 (Observational vs Experiment)
[퀴즈] 연구자가 흡연과 폐암의 관계를 알아보기 위해, 지난 10년간의 병원 기록을 수집하여 분석했습니다.
이 연구는 실험(Experiment)일까요, 관찰 연구(Observational Study)일까요?
정답 확인
정답: 관찰 연구 (Observational Study)
연구자가 사람들에게 강제로 담배를 피우게(개입/Treatment) 하지 않고, 단순히 기록을 관찰했기 때문입니다.
따라서 "흡연이 폐암의 원인이다"라고 단정 짓는 인과관계 추론에는 주의해야 합니다.
2. 수치형 데이터 요약 (Numerical Data)
Exercise 2.1 히스토그램과 비대칭도 (Skewness)
[문제] 소득(Income) 데이터는 일반적으로 오른쪽 꼬리가 긴(Right-skewed) 분포를 보입니다.
파이썬으로 이를 시각화하고 확인해 봅시다.
import matplotlib.pyplot as plt import numpy as np # 가상 데이터: 1000명의 소득 (오른쪽으로 치우친 분포 생성) income = np.random.exponential(scale=50000, size=1000) plt.figure(figsize=(8, 4)) # [미션] bins 개수를 50으로 늘려서 더 자세히 관찰해 보세요. plt.hist(income, bins=20, color='skyblue', edgecolor='black') plt.title("Distribution of Income (Right Skewed)") plt.xlabel("Income ($)") plt.ylabel("Frequency") plt.show() print(">> 해석: 왼쪽(저소득)에 데이터가 몰려있고, 오른쪽(고소득)으로 꼬리가 길게 뻗어 있습니다.")
Exercise 2.2 박스플롯과 이상치(Outlier) 탐지
[문제] 두 학급(Class A, Class B)의 성적을 비교하려고 합니다.
Boxplot을 그려서 어느 반의 성적이 더 안정적인지, 이상치는 없는지 확인하시오.
import seaborn as sns import pandas as pd # 데이터 생성 data = { 'Score': [60, 65, 70, 75, 80, 85, 90, 55, 100, 30] + [70, 72, 74, 76, 78, 80, 75, 77, 73, 79], 'Class': ['A']*10 + ['B']*10 } df = pd.DataFrame(data) # [미션] 박스플롯 그리기 sns.boxplot(x='Class', y='Score', data=df, palette="Set3") plt.title("Class A vs Class B") plt.show()
결과 해석하기
- Class A: 상자(Box)가 길고 점(이상치)이 보일 수 있습니다. 성적 편차가 큽니다.
- Class B: 상자가 짧고 모여 있습니다. 성적이 고르게 분포합니다.
3. 범주형 데이터 요약 (Categorical Data)
Exercise 2.3 분할표(Contingency Table) 만들기
[문제] 타이타닉 데이터에서 '좌석 등급(Class)'과 '생존 여부(Survived)'의 관계를 보려고 합니다.
pd.crosstab을 사용하여 표를 만들어 보세요.
import pandas as pd # 가상 데이터 (1=생존, 0=사망) data = { 'Class': ['1st', '1st', '1st', '2nd', '2nd', '3rd', '3rd', '3rd', '3rd', '3rd'], 'Survived': [1, 1, 0, 1, 0, 0, 0, 0, 1, 0] } df = pd.DataFrame(data) # [미션] 분할표 생성 (margins=True를 넣으면 합계도 나옵니다) table = pd.crosstab(df['Class'], df['Survived'], margins=True) print(table)
Exercise 2.4 비율 비교 시각화 (Bar Plot)
[문제] 위에서 만든 분할표를 눈으로 비교하기 쉽게 막대그래프로 그려봅시다.
# 위에서 만든 데이터를 시각화 table_plot = pd.crosstab(df['Class'], df['Survived']) table_plot.plot(kind='bar', stacked=True, color=['red', 'green']) plt.title("Survival by Class") plt.ylabel("Count") plt.show() print(">> 해석: 1st 클래스(맨 왼쪽)의 초록색(생존) 비율이 다른 클래스보다 높은가요?")
4. 두 변수의 관계 (Correlation)
Exercise 4.1 상관계수 직접 계산하기
[문제] 공부 시간(Time)과 성적(Score)의 관계입니다.
1. 산점도(Scatter plot)를 그리고
2. 상관계수(Correlation Coefficient)를 구하시오.
import numpy as np import matplotlib.pyplot as plt time = [2, 4, 6, 8, 10] score = [50, 60, 75, 80, 95] # [미션] 상관계수 계산 corr = np.corrcoef(time, score)[0, 1] plt.scatter(time, score, color='purple') plt.title(f"Study Time vs Score (r = {corr:.2f})") plt.xlabel("Hours") plt.ylabel("Score") plt.grid(True) plt.show()
