새로운 시각 2.1 — 산점도의 역사적 의의

산점도는 19세기 프랜시스 골턴(Francis Galton)에 의해 체계화된 통계 시각화 도구이다. 골턴은 부모와 자녀의 키 관계를 연구하면서 산점도를 활용하여 "평균으로의 회귀(regression to the mean)" 현상을 발견했다. 이는 현대 회귀분석의 시초가 되었다.

산점도를 해석할 때 기억해야 할 핵심 원칙:

  1. 상관관계 ≠ 인과관계: 두 변수가 함께 움직인다고 해서 하나가 다른 하나를 야기하는 것은 아니다.
  2. 맥락의 중요성: 통계적 패턴은 반드시 연구 배경과 함께 해석해야 한다.
  3. 제3의 변수: 숨겨진 교란변수가 관찰된 관계를 설명할 수 있다.

import numpy as np import matplotlib.pyplot as plt # 아래 값을 'linear', 'quadratic', 'none', 'exponential' 중 하나로 변경해 보라 relationship = 'linear' np.random.seed(42) x = np.random.uniform(0, 10, 100) noise = np.random.normal(0, 2, 100) if relationship == 'linear': y = 2 * x + 3 + noise title = '선형 관계 (y = 2x + 3)' elif relationship == 'quadratic': y = 0.5 * x**2 - 3*x + 10 + noise title = '이차 관계 (y = 0.5x² - 3x + 10)' elif relationship == 'exponential': y = 2 * np.exp(0.3 * x) + noise * 3 title = '지수 관계 (y = 2·e^(0.3x))' else: y = np.random.normal(5, 3, 100) title = '무관계 (랜덤)' corr = np.corrcoef(x, y)[0,1] plt.figure(figsize=(8, 5)) plt.scatter(x, y, alpha=0.6, edgecolors='black') plt.title(f'{title} | 상관계수 r = {corr:.3f}') plt.xlabel('X'); plt.ylabel('Y') plt.grid(True, alpha=0.3) plt.show()

2.1.2 점도표와 평균

때때로 두 변수는 너무 많아 하나의 변수만 관심 대상이 될 수 있다. 이런 경우, 점도표(dot plot)가 가장 기본적인 표시 방법이다.

평균(mean)은 표본의 산술 평균으로, 관측값을 모두 더한 후 관측치 수로 나눈 값이다.

x̄ = (x₁ + x₂ + ··· + xₙ) / n

여기서 x̄는 "x-bar"로 읽으며 표본평균을 나타낸다. 개별 관측값은 x₁, x₂, …, xₙ으로, 총 n개의 관측치가 있다.

새로운 시각 2.2 — 모수와 통계량: 그리스 문자 vs 라틴 문자

통계학에서 모수(parameter)와 통계량(statistic)은 기호 체계부터 구분된다:

• 모수 (모집단의 참값): μ(평균), σ(표준편차), p(비율) — 그리스 문자

• 통계량 (표본에서 계산): x̄(평균), s(표준편차), p̂(비율) — 라틴 문자

이 기호 체계 자체가 '알려진 것(추정 대상)'과 '계산으로 추정하는 것'의 본질적 차이를 반영한다.

새로운 시각 2.3 — 평균의 물리적 의미

평균을 이해하는 직관적인 방법은 무게중심(center of mass)으로 생각하는 것이다. 데이터를 수평 막대 위에 놓인 무게추라고 상상하자. 평균은 막대가 균형을 이루는 지점(받침점)이다.

이 물리적 해석은 왜 이상치가 평균에 큰 영향을 미치는지 설명한다: 받침점에서 멀리 떨어진 무게추는 더 큰 레버 효과를 가지므로, 균형점을 자신 쪽으로 끌어당긴다.

수학적으로, 평균은 편차의 합이 0이 되는 유일한 점이다: Σ(xᵢ − x̄) = 0

2.1.3 히스토그램과 분포의 형태

분포의 형태: 비대칭성

데이터가 한쪽으로 길게 뻗어 있을 때 분포는 비대칭(skewed)이라 한다.

오른쪽 치우침(Right skewed): 오른쪽 꼬리가 길다 (양의 비대칭)  |  왼쪽 치우침(Left skewed): 왼쪽 꼬리가 길다  |  대칭(Symmetric): 양쪽이 비슷
긴 꼬리로 비대칭성 파악하기: 데이터가 한 방향으로 길게 뻗어 있으면 분포에 긴 꼬리가 있다. 왼쪽 꼬리가 길면 왼쪽 치우침, 오른쪽 꼬리가 길면 오른쪽 치우침이다.

새로운 시각 2.4 — 히스토그램의 구간 너비 선택

히스토그램에서 구간 너비(bin width)의 선택은 데이터 해석에 큰 영향을 미친다.

구간이 너무 좁으면: 노이즈가 많아져 패턴 파악이 어렵다.
구간이 너무 넓으면: 중요한 세부사항이 사라진다.

경험 법칙으로 Sturges' Rule k = 1 + log₂(n), Freedman-Diaconis Rule h = 2·IQR·n^(-1/3) 이 있다. 이러한 규칙은 시작점일 뿐이다.

2.1.4 분산과 표준편차

평균이 데이터의 중심을 설명하는 방법으로 소개되었다면, 산포도(variability)도 중요하다. 여기서 두 가지 산포도 측정치를 소개한다: 분산(variance)과 표준편차(standard deviation).

표준편차는 대략적으로 관측값이 평균에서 얼마나 떨어져 있는지를 설명한다.

편차, 분산, 표준편차 계산

편차ᵢ = xᵢ − x̄     s² = Σ(xᵢ − x̄)² / (n−1)     s = √s²
왜 n−1로 나누는가? n 대신 n−1로 나누는 것을 베셀의 수정(Bessel's correction)이라 하며, 표본분산이 모분산의 불편추정량이 되도록 한다.

새로운 시각 2.5 — 체비셰프 부등식과 경험적 규칙

"70%와 95% 규칙"은 경험적 규칙(Empirical Rule, 68-95-99.7 규칙)이라 불리며, 정규분포에 적용된다.

하지만 모든 분포에 적용되는 더 일반적인 결과가 있다: 체비셰프의 부등식

P(|X − μ| ≥ kσ) ≤ 1/k²

비교표:

k경험적 규칙 (정규)체비셰프 (모든 분포)
1σ~68%0% (보장 없음)
2σ~95%≥75%
3σ~99.7%≥89%

2.1.5 상자그림, 사분위수, 중앙값

상자그림(box plot)은 다섯 가지 통계량을 사용하여 데이터셋을 요약하면서 비정상적인 관측값도 표시한다.

중앙값

중앙값(median)은 데이터를 절반으로 나눈다. 데이터가 가장 작은 값에서 가장 큰 값으로 정렬되면, 중앙값은 정확히 중간에 있는 관측값이다. 관측치 수가 짝수이면 두 중간값의 평균이다.

PYTHON — 중앙값과 사분위수 계산

사분위범위(IQR)와 이상치

IQR = Q₃ − Q₁     이상치 경계: Q₁ − 1.5·IQR ~ Q₃ + 1.5·IQR

수염(whiskers)은 상자 바깥의 데이터를 포착하려 하지만, 길이는 1.5×IQR을 초과할 수 없다. 수염을 넘어서는 관측값은 점으로 표시되며, 이를 이상치(outliers)라 부른다.

새로운 시각 2.8 — 심슨의 역설 (Simpson's Paradox)

전체 데이터에서 관찰되는 경향이 데이터를 그룹으로 나눌 때 역전되는 현상이다.

예시: 1973년 UC Berkeley 대학원 입학

전체: 남성 합격률 44%, 여성 35% → 성별 차별처럼 보인다.

하지만 학과별로 보면 대부분의 학과에서 여성 합격률이 같거나 높았다! 여성이 경쟁이 치열한 학과에 더 많이 지원했기 때문이다.

교훈: 항상 잠재적인 교란변수(confounding variable)를 고려해야 한다.

PYTHON — 심슨의 역설 시뮬레이션
import pandas as pd data = { '학과': ['A', 'A', 'B', 'B'], '성별': ['남', '여', '남', '여'], '지원자': [825, 108, 560, 25], '합격자': [512, 89, 353, 17] } df = pd.DataFrame(data) df['합격률'] = df['합격자'] / df['지원자'] print("학과별 합격률:") print(df.pivot(index='학과', columns='성별', values='합격률').round(3)) total = df.groupby('성별').agg({'지원자': 'sum', '합격자': 'sum'}) total['전체_합격률'] = total['합격자'] / total['지원자'] print("\n전체 합격률:") print(total['전체_합격률'].round(3)) print("\n→ 학과별로는 여성 합격률이 높지만, 전체로 합치면 역전된다!")

2.3.3 독립성 검정

시뮬레이션을 10,000번으로 확대하여 우연만으로 발생하는 차이의 분포를 정밀하게 파악한다.

PYTHON — 10,000회 시뮬레이션과 p-값 계산
import numpy as np import matplotlib.pyplot as plt def simulate_trial(): patients = ['infected'] * 11 + ['not_infected'] * 9 np.random.shuffle(patients) vaccine = patients[:14]; placebo = patients[14:] return placebo.count('infected')/6 - vaccine.count('infected')/14 np.random.seed(42) n_sim = 10000 sim_diffs = [simulate_trial() for _ in range(n_sim)] plt.figure(figsize=(12, 6)) plt.hist(sim_diffs, bins=30, edgecolor='black', alpha=0.7, density=True) plt.axvline(0.643, color='red', linestyle='--', linewidth=2, label='관찰된 차이: 64.3%') plt.axvline(0, color='green', linestyle='-', linewidth=2, label='기대값: 0') plt.xlabel('감염률 차이 (위약 - 백신)') plt.ylabel('밀도') plt.title(f'{n_sim:,}번 시뮬레이션에서의 감염률 차이 분포') plt.legend() plt.grid(True, alpha=0.3) plt.show() p_value = sum(1 for d in sim_diffs if d >= 0.643) / n_sim print(f"p-값: {p_value:.4f} ({p_value*100:.2f}%)") print(f"\n{'='*50}") if p_value < 0.05: print("결론: H₀를 기각한다.") print("백신이 감염률 감소에 효과적이라는 통계적 증거가 있다.") else: print("결론: H₀를 기각할 수 없다.")

새로운 시각 2.9 — 통계적 추론의 기초: 시뮬레이션에서 공식으로

이 사례 연구는 통계적 추론의 핵심 아이디어를 소개한다:

  1. 귀무가설(H₀): "효과가 없다"는 기본 가정
  2. 대립가설(Hₐ): 우리가 증거를 찾고자 하는 주장
  3. p-값: H₀가 참일 때 관찰된 결과가 발생할 확률
  4. 유의수준: H₀를 기각하기 위한 임계값 (일반적으로 0.05)

중요한 점:

  • 통계적으로 유의하다고 해서 실용적으로 중요하다는 것을 의미하지는 않는다
  • p-값이 작다고 해서 효과 크기가 크다는 것을 의미하지 않는다
  • 통계적 검정은 인과관계를 증명하지 않는다 (단, 무작위 실험의 경우 인과 추론이 가능)

제5장 이후에서 이러한 개념을 더 형식적으로 다룬다.

import numpy as np from scipy import stats # 원하는 데이터를 입력하라 my_data = [23, 45, 12, 67, 34, 89, 56, 78, 90, 11, 33, 55, 77, 99, 44] data = np.array(my_data) print(f"데이터: {sorted(my_data)}") print(f"{'='*40}") print(f"관측치 수: {len(data)}") print(f"평균: {np.mean(data):.2f}") print(f"중앙값: {np.median(data):.2f}") print(f"분산: {np.var(data, ddof=1):.2f}") print(f"표준편차: {np.std(data, ddof=1):.2f}") print(f"Q1: {np.percentile(data, 25):.2f}") print(f"Q3: {np.percentile(data, 75):.2f}") print(f"IQR: {np.percentile(data, 75) - np.percentile(data, 25):.2f}") print(f"최솟값: {data.min()}") print(f"최댓값: {data.max()}") print(f"범위: {data.max() - data.min()}") print(f"비대칭도: {stats.skew(data):.2f}") # 이상치 확인 Q1 = np.percentile(data, 25); Q3 = np.percentile(data, 75) IQR = Q3 - Q1 outliers = data[(data < Q1-1.5*IQR) | (data > Q3+1.5*IQR)] print(f"이상치: {list(outliers) if len(outliers) > 0 else '없음'}")
