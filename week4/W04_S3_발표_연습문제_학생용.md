### 해답 11. (보너스) 정규화와 표준화

```python
prices = np.array([50, 80, 120, 200, 500, 75, 90, 150, 300, 85])

# (a)
min_max = (prices - prices.min()) / (prices.max() - prices.min())
print(f"Min-Max: {min_max.round(3)}")

# (b)
z_score = (prices - prices.mean()) / prices.std()
print(f"Z-Score: {z_score.round(3)}")

# (c)
# Min-Max: $500은 1.0으로 변환되며, 나머지가 0에 가깝게 압축됨.
# Z-Score: $500은 약 +2.x의 높은 양수가 되며, 다른 값들은 음수.
# → 이상치($500)가 Min-Max에서는 전체 스케일을 왜곡하고,
#   Z-Score에서는 상대적 위치로 표현되어 이상치 식별이 더 용이하다.
```

## 4주차 핵심 정리

| 개념 | 핵심 |
|---|---|
| **넘파이**(NumPy) | for문 없이 배열 전체를 한꺼번에 연산한다 |
| **벡터화**(Vectorization) | `arr * 2`, `arr[arr > 100]` — 50~100배 속도 향상 |
| **브로드캐스팅**(Broadcasting) | 크기 다른 배열도 자동 확장하여 연산 |
| **판다스**(Pandas) | 표 형태 데이터를 다루는 핵심 도구 |
| **데이터프레임**(DataFrame) | 2차원 표, CSV 입출력, 조건 필터링, groupby |
| 이상치 처리 | 평균 vs 중앙값, 극단값 제거 전후 비교가 중요하다 |
