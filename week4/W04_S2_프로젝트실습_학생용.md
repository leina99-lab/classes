## 단계 2: 타겟 필터링 — 2**베드룸**(Bedroom) 추출

우리의 미션은 "방이 둘 딸린 집"이다. 그런데 이 데이터에는 `bedrooms` 열이 없다. `name` 열에서 "2 bed", "2 BR", "2 bedroom" 등의 키워드로 검색하거나, `room_type`이 "Entire home/apt"인 것을 기준으로 분석한다.

```python
# 2-1) room_type이 "Entire home/apt"인 매물만 필터링
entire = df[df["room_type"] == "Entire home/apt"].copy()
print(f"Entire home/apt: {len(entire):,}건")
print(f"전체 대비 비율: {len(entire)/len(df)*100:.1f}%")
```

```python
# 2-2) name 열에서 "2" + ("bed" 또는 "br" 또는 "bedroom") 포함 여부 검색
# 대소문자 무시 검색
mask_2bed = entire["name"].str.lower().str.contains(
    r"2\s*(bed|br|bedroom)", regex=True, na=False
)
bed2 = entire[mask_2bed].copy()
print(f"2베드룸 추정: {len(bed2):,}건")
```

```python
# 2-3) 필터링 결과 기초 통계
print(f"\n=== 2베드룸 가격 통계 ===")
print(f"  건수:   {len(bed2):,}")
print(f"  평균:   ${bed2['price'].mean():.0f}")
print(f"  중앙값: ${bed2['price'].median():.0f}")
print(f"  최소:   ${bed2['price'].min()}")
print(f"  최대:   ${bed2['price'].max()}")
print(f"  표준편차: ${bed2['price'].std():.0f}")
```

## 단계 4: 지역별 가격 통계 — `groupby` 활용

```python
# 4-1) 지역별(neighbourhood_group) 가격 통계
region_stats = bed2.groupby("neighbourhood_group")["price"].agg(
    ["count", "mean", "median", "min", "max", "std"]
).round(0)

region_stats.columns = ["건수", "평균", "중앙값", "최소", "최대", "표준편차"]
print("=== 지역별 2베드룸 가격 통계 ===")
print(region_stats)
```

```python
# 4-2) 지역별 건수 비율
region_stats["비율(%)"] = (region_stats["건수"] / region_stats["건수"].sum() * 100).round(1)
print("\n=== 지역별 비율 포함 ===")
print(region_stats.sort_values("건수", ascending=False))
```

## 단계 5: **이상치**(Outlier) 제거 전후 비교

가격이 극단적으로 높은 매물(럭셔리 펜트하우스 등)은 "적정 가격" 산출에 방해가 된다.

```python
# 5-1) 이상치 확인
print("=== 이상치 탐색 ===")
print(f"  $0 매물: {len(bed2[bed2['price'] == 0])}건")
print(f"  $1,000 초과: {len(bed2[bed2['price'] > 1000])}건")
print(f"  $5,000 초과: {len(bed2[bed2['price'] > 5000])}건")
```

```python
# 5-2) 이상치 제거: 가격 $0 제외, $1,000 초과 제외
bed2_clean = bed2[(bed2["price"] > 0) & (bed2["price"] <= 1000)].copy()
print(f"\n이상치 제거 전: {len(bed2):,}건")
print(f"이상치 제거 후: {len(bed2_clean):,}건")
print(f"제거된 건수: {len(bed2) - len(bed2_clean):,}건 ({(len(bed2)-len(bed2_clean))/len(bed2)*100:.1f}%)")
```

```python
# 5-3) 이상치 제거 전후 비교
print("\n=== 이상치 제거 전후 비교 ===")
print(f"{'':>10} {'제거 전':>10} {'제거 후':>10} {'변화':>10}")
print("-" * 45)

for stat_name, before, after in [
    ("평균", bed2["price"].mean(), bed2_clean["price"].mean()),
    ("중앙값", bed2["price"].median(), bed2_clean["price"].median()),
    ("표준편차", bed2["price"].std(), bed2_clean["price"].std()),
    ("최대", bed2["price"].max(), bed2_clean["price"].max()),
]:
    diff = after - before
    print(f"{stat_name:>10} ${before:>8,.0f} ${after:>8,.0f} ${diff:>+8,.0f}")
```

```python
# 5-4) 이상치 제거 후 지역별 재계산
region_clean = bed2_clean.groupby("neighbourhood_group")["price"].agg(
    ["count", "mean", "median"]
).round(0)
region_clean.columns = ["건수", "평균", "중앙값"]
print("\n=== 이상치 제거 후 지역별 ===")
print(region_clean.sort_values("중앙값", ascending=False))
```

**📝 관찰을 기록하라:**

- 평균과 중앙값 중 어느 것이 "적정 가격"을 더 잘 대표하는가? 이유는?

## 단계 6: 검증 — 필터링 과정 요약

분석 과정에서 데이터가 얼마나 줄었는지 전체 흐름을 검증하라.

```python
# 전체 파이프라인 요약
print("=" * 50)
print("  데이터 필터링 파이프라인 요약")
print("=" * 50)
steps = [
    ("원본 데이터", len(df)),
    ("Entire home/apt만", len(entire)),
    ("2베드룸 추정", len(bed2)),
    ("이상치 제거 후", len(bed2_clean)),
]
for i, (name, count) in enumerate(steps):
    bar = "█" * int(count / len(df) * 30)
    pct = count / len(df) * 100
    print(f"  {i+1}. {name:<15} {count:>6,}건 ({pct:>5.1f}%) {bar}")
print("=" * 50)
```

## 단계 7: 최종 가격 제안

```python
# 최종 분석 결과
print("\n" + "=" * 50)
print("  📊 최종 가격 제안")
print("=" * 50)

for region in bed2_clean["neighbourhood_group"].unique():
    subset = bed2_clean[bed2_clean["neighbourhood_group"] == region]
    if len(subset) >= 5:   # 최소 5건 이상일 때만
        q1 = subset["price"].quantile(0.25)
        q3 = subset["price"].quantile(0.75)
        med = subset["price"].median()
        print(f"\n  [{region}]")
        print(f"    추천 범위: ${q1:.0f} ~ ${q3:.0f}")
        print(f"    적정 가격(중앙값): ${med:.0f}")
        print(f"    분석 기반 건수: {len(subset):,}건")

print("\n" + "=" * 50)
print("  💡 결론: 지역을 선택한 후, 해당 지역의 중앙값을")
print("     기준으로 Q1~Q3 범위에서 가격을 설정하는 것이")
print("     데이터에 기반한 합리적 전략이다.")
print("=" * 50)
```
