PART A. 핵심 개념 정리

A1. 변수(Variable)란?

데이터를 저장하고 재사용하기 위한 이름이 붙은 저장 공간이다.

= 는 "같다"가 아니라 "저장하라"는 뜻이다.  |  "같다"는 == 를 사용한다.

A2. 자료형(Data Type) 4가지

자료형설명예시
int정수(Integer)42, -7, 0
float실수(Floating Point)3.14, -0.5, 1e3
str문자열(String)"Hello", 'Python'
bool불리언(Boolean)True, False

A3. 숫자형 연산자

연산자이름예시결과
+더하기10 + 313
-빼기10 - 37
*곱하기10 * 330
/나누기 (항상 float)10 / 33.333…
//몫(Floor Division)10 // 33
%나머지(Modulo)10 % 31
**거듭제곱(Power)10 ** 31000

A4. 문자열(String) 핵심

슬라이싱 s[시작:끝:간격] — 끝 인덱스는 포함하지 않는다.
f-string: f"{변수명}" — 중괄호 안 변수가 자동으로 삽입된다.
# A. 개념 확인 — 직접 실행해보라 print("=== 자료형 확인 ===") print(type(621328)) # int print(type(3.14)) # float print(type("코로나")) # str print(type(True)) # bool print("\n=== 문자열 기차 ===") s = "COVID-19" print(s[0], s[-1], s[0:5], s[::-1]) print("\n=== f-string ===") country = "한국" cases = 621328 print(f"{country}의 신규 확진자: {cases:,}명")

PART B. 핑퐁 코딩 (Ping-Pong Coding)

Ping 교수자가 시연한다 → Pong 학생이 즉시 변형 과제를 작성한다.

빈칸 ___ 을 채워야 코드가 실행된다. 정답을 먼저 보지 말고 반드시 스스로 시도한 후 확인하라.

Ping 1 변수와 자료형 — 교수자 시연

# 오늘 서울의 기온을 변수에 저장하자 city = "서울" temperature = 3.5 print(f"오늘 {city}의 기온은 {temperature}도이다.") print(f"city의 자료형: {type(city)}") print(f"temperature의 자료형: {type(temperature)}")

# Pong 1 — 빈칸을 채워라 date = "2022-03-17" country = ___ # "한국"을 저장 new_cases = ___ # 621328을 저장 print(f"{date} {country}의 신규 확진자 수: {new_cases}명") print(f"new_cases의 자료형: {type(new_cases)}")

출력:

1주차 수업 후 연습문제 10문항의 코드 + 출력 + 검증을 제출하라.
