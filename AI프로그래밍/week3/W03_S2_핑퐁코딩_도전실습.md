## Ping-Pong ②: **for문** + `enumerate()` → 확진자 변화율 계산

### 🏓 Ping (교수자 시연)

```python
# 일일 매출과 전일 대비 변화율 계산
sales = [100, 120, 90, 150, 130]

for i in range(1, len(sales)):
    change = (sales[i] - sales[i-1]) / sales[i-1] * 100
    arrow = "↑" if change > 0 else "↓"
    print(f"  {i+1}일차: {sales[i]}만원 ({arrow} {abs(change):.1f}%)")
```

        return ___
    except ___:
        return default

clean_data = []

    result = safe_convert(val, default=-1)
