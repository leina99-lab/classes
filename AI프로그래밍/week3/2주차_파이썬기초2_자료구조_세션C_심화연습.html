Debug ① 리스트 — 3곳 수정

힌트: True 대소문자, 슬라이싱 범위, 내장 함수 vs 메서드

cases = [523, 681, 445, 892, 1023] cases.sort(reverse=true) # 오류 1 top3 = cases[1:3] # 오류 2 print("상위 3개국:", top3) total = cases.sum() # 오류 3 print("총 확진자:", total)
▶ 정답 확인 
cases.sort(reverse=True)    # True (대문자)
top3 = cases[:3]            # [0:3] 또는 [:3]
total = sum(cases)          # 내장함수 sum()

Debug ② 딕셔너리 — 3곳 수정

힌트: 딕셔너리 접근 방식, 메서드 이름 단수/복수
