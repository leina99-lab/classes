games.csv — 보드게임 데이터로 배우는 조건문과 반복문

학습 목표
1. if/elif/else 로 게임 평점 등급을 판별한다.
2. 비교·논리 연산자 로 복합 조건을 처리한다.
3. while 로 게임 목록을 반복 처리한다.
4. break / continue 로 반복 흐름을 제어한다.
5. 조건부 표현식 으로 코드를 압축한다.

1if / elif / elsePong 1~6

3while 기본Pong 11~14
4break / continuePong 15~18

개념 설명
if 조건:
    블록1
elif 조건2:

  • 명작: 평점 8.0 이상

  • 보통: 6.0 이상
