학습 목표
1. 리스트(List) 로 고객 점수를 관리하고 정렬·필터링한다.

Section 1. 리스트 (List)

개념 설명

리스트(list)는 순서가 있는 변경 가능한 자료구조이다.

scores = [39, 81, 6, 77, 40, 76, 6, 94, 3, 72]
scores[0]      # 39  (인덱싱)
scores[1:4]    # [81, 6, 77]  (슬라이싱)
scores.append(55)   # 추가
scores.remove(6)    # 삭제 (첫 번째)
sorted(scores)      # 정렬 (원본 유지)
scores.sort()       # 정렬 (원본 변경)

Section 2. 튜플 (Tuple)

개념 설명

튜플(tuple)은 순서 있고 변경 불가(immutable)한 자료구조이다.

customer = (1, 'Male', 19, 15, 39)  # (ID, 성별, 나이, 수입, 점수)
customer[0]      # 1 (읽기만 가능)
customer[0] = 2  # TypeError! 수정 불가
id, genre, age, income, score = customer  # 언패킹
변경되면 안 되는 데이터(좌표·등록 정보)에 튜플을 사용한다.
🔵 Ping 7 — 강사 시범
강사 시범: 튜플 생성·언패킹
🟡 Pong 7 — 학생 실습
각 고객의 (id, genre, score) 튜플 리스트를 만들고 score 기준 정렬하라.
▶ 정답 보기 
tuples = [(p['id'], p['genre'], p['score']) for p in customers]
ranked = sorted(tuples, key=lambda x: x[2], reverse=True)
for rank, (cid, genre, score) in enumerate(ranked, 1):
    print(f'{rank:2}위: 고객{cid:3d} ({genre}) {score}점')
🔵 Ping 8 — 강사 시범
강사 시범: 튜플 불변성 확인
🟡 Pong 8 — 학생 실습
고객 5명의 (income, score) 쌍을 튜플 리스트로 만들고 income 기준 내림차순 정렬하라.
▶ 정답 보기 
pairs = [(p['income'], p['score']) for p in customers]
ranked = sorted(pairs, key=lambda x: x[0], reverse=True)
for inc, sc in ranked:
    print(f'수입 {inc}k$: 소비점수 {sc}')
🔵 Ping 9 — 강사 시범
강사 시범: 다중 반환값 → 튜플 언패킹
🟡 Pong 9 — 학생 실습
연수입 리스트의 (최솟값, 최댓값, 평균, 범위) 튜플을 반환하는 함수를 만들어라.
▶ 정답 보기 
def income_stats(data):
    return min(data), max(data), sum(data)/len(data), max(data)-min(data)

incomes = [p['income'] for p in customers]
lo, hi, avg, rng = income_stats(incomes)
print(f'최소:{lo}k$, 최대:{hi}k$, 평균:{avg:.1f}k$, 범위:{rng}k$')
🔵 Ping 10 — 강사 시범
강사 시범: zip → 튜플 쌍 생성
🟡 Pong 10 — 학생 실습
zip으로 (id, age, income, score) 4-튜플 리스트를 만들어 나이 기준 내림차순 정렬하라.
▶ 정답 보기 
quads = list(zip(
    [p['id'] for p in customers],
    [p['age'] for p in customers],

for q in sorted(quads, key=lambda x: x[1], reverse=True):
    print(f'고객{q[0]:3d}: {q[1]}세  수입{q[2]}k$  점수{q[3]}')

Section 3. 딕셔너리 (Dictionary)

개념 설명

딕셔너리(dict)는 키(key)-값(value) 쌍의 자료구조이다.

customer = {'id':1, 'genre':'Male', 'age':19, 'income':15, 'score':39}
customer['age']           # 19 (접근)
customer['vip'] = True   # 키 추가
del customer['vip']       # 키 삭제
customer.get('tier','Basic')  # 없으면 기본값
customer.keys()   # 키 목록
customer.values() # 값 목록
customer.items()  # (키,값) 쌍 목록

▶ 정답 보기 
result = {}

    group = [p for p in customers if p['genre'] == g]

for g, v in result.items():

Section 4. 집합 (Set)

개념 설명

집합(set)은 순서 없고 중복 없는 자료구조이다.

{1, 2, 3} | {2, 3, 4}   # 합집합 → {1,2,3,4}
{1, 2, 3} & {2, 3, 4}   # 교집합 → {2,3}
{1, 2, 3} - {2, 3, 4}   # 차집합 → {1}
{1,2,3} <= {1,2,3,4}    # 부분집합 → True
🔵 Ping 16 — 강사 시범
강사 시범: 집합 연산으로 고객 그룹 분석
🟡 Pong 16 — 학생 실습
수입 17k$ 이상과 소비점수 70 이상인 고객 ID 집합을 만들고 교집합·합집합·차집합을 구하라.
▶ 정답 보기 
hi_inc  = {p['id'] for p in customers if p['income'] >= 17}
hi_sc   = {p['id'] for p in customers if p['score'] >= 70}
print('고수입:', hi_inc)
print('고점수:', hi_sc)
print('교집합(둘 다):', hi_inc & hi_sc)
print('합집합(하나라도):', hi_inc | hi_sc)
print('고수입이지만 저점수:', hi_inc - hi_sc)
🔵 Ping 17 — 강사 시범
강사 시범: set으로 중복 제거 & 빈도 분석
🟡 Pong 17 — 학생 실습

▶ 정답 보기

ratio   = len(unique) / len(incomes) * 100

🟡 Pong 18 — 학생 실습

▶ 정답 보기

Section 5. 불리언 (Boolean) & copy()

개념 설명

불리언(bool) — True / False

bool(0)   # False   bool(1)   # True
bool('')  # False   bool([])  # False
bool(39)  # True    bool([1]) # True

얕은 복사 vs 깊은 복사

import copy
a = [1, 2, 3]
b = a           # 같은 객체 참조
c = a.copy()    # 얕은 복사 → 새 객체
d = copy.deepcopy(a)  # 깊은 복사
🔵 Ping 19 — 강사 시범
강사 시범: 불리언 all()/any()
🟡 Pong 19 — 학생 실습
all()과 any()를 이용해 ①모든 고객이 25세 이상인지 ②VIP(80+) 고객이 존재하는지 ③30대 이상 고객이 절반 이상인지 확인하라.
▶ 정답 보기 
all_25    = all(p['age'] >= 25 for p in customers)
has_vip   = any(p['score'] >= 80 for p in customers)
over30    = sum(1 for p in customers if p['age'] >= 30)
half_30   = over30 >= len(customers) / 2
print(f'모두 25세 이상: {all_25}')
print(f'VIP 고객 존재: {has_vip}')
print(f'30대+ 절반 이상: {half_30} ({over30}/{len(customers)}명)')
🔵 Ping 20 — 강사 시범
강사 시범: copy()로 안전한 데이터 수정
🟡 Pong 20 — 학생 실습
customers 리스트를 깊은 복사하고, 복사본에서 소비점수 50 미만을 50으로 조정한 뒤 원본과 비교하라.
▶ 정답 보기 
import copy
adjusted = copy.deepcopy(customers)
for p in adjusted:
    if p['score'] < 50:
        p['score'] = 50

print(f'{"ID":^5} {"원본":>6} {"수정":>6}')
print('-' * 18)
for orig, adj in zip(customers, adjusted):
    changed = ' *' if orig['score'] != adj['score'] else ''
    print(f'{orig["id"]:^5} {orig["score"]:>6} {adj["score"]:>6}{changed}')
