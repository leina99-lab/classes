▶ 버튼을 눌러 데이터가 층을 통과하는 과정을 관찰하라.

2. 순전파 수식 요약

각 층에서 일어나는 연산:

1단계 — 선형 변환: z = W·x + b   (행렬곱 + 편향)
2단계 — 활성화 함수: a = f(z)   (비선형성 부여)

입력 → [W₁·x+b₁] → f → [W₂·a₁+b₂] → f → ... → 출력
→ AI 연결: GPT-4는 이 순전파를 수백 개의 층에서 수행한다. 각 층에는 수십억 개의 가중치(W)가 있으며, 입력 토큰이 이 층들을 차례로 통과하면서 점점 더 추상적인 표현으로 변환된다. 한 번의 순전파로 다음 단어를 예측한다.
